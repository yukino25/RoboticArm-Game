Robotic Arm Game
================
1. Extract this entire folder before running (do not run game.exe from inside the zip).
2. Double-click game.exe to play.

Controls:
  A / D       — rotate arm
  W / S       — extend / retract arm / move track
  Q / E       — switch active arm
  Space       — grab / release object
  R           — restart level
  Escape      — menu
